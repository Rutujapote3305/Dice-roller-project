# Dynamic Dice Roller
A lightweight, interactive web application that allows users to roll multiple dice simultaneously. Built with Vanilla JavaScript, this project focuses on dynamic UI updates and random number generation.


## Features
**Adjustable Dice Count:** Users can specify how many dice they want to roll.
**Dynamic Image Rendering:** Automatically displays the correct dice face (1-6) for every result.
**Text Summary:** Provides a comma-separated list of the roll results for quick reading.
**Responsive Design:** The layout adapts to show multiple dice in a clean grid/row format.

## Built With
**HTML5:** Semantic structure.
**CSS3:** Custom styling with Flexbox for layout and box-shadows for depth.
**JavaScript (ES6):** Logic for random number generation and DOM manipulation.
